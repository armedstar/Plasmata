*-Overview-*

P L A S M A T A is an oscillator with a noise modulation system designed around a single idea:

xx Distorted perception xx

Signal, noise, and time interact to produce evolving, unstable, and living textures.

----------------------------------------------------------------------------------

*-Features-*

– Core oscillator with frequency adjustment and selectable waveforms:
   • Sine
   • Saw
   • Square
   • Triangle

- Noise modulation:
   • Off
   • White
   • Pink
   • Red
   • Black

- Built-in delay for temporal smearing and echo
   • Adjustable time and feedback
   • Trigger input that can retrigger the delay start

Designed for organic movement and evolving tones

----------------------------------------------------------------------------------

*-Installation-*

Build from source (CMake)

1. Clone this repository:
   git clone https://github.com/YOUR_USERNAME/plasmata.git
   cd plasmata

2. Point CMake to your Rack SDK location (if needed):
   export RACK_DIR=/path/to/Rack-SDK

3. Configure and build:
   cmake -S . -B build
   cmake --build build -j

4. Install the built plugin to your Rack plugins directory:
   ./install.sh

5. Restart VCV Rack.
